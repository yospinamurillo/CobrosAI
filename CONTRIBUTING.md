# Contributing to CobrosAI

Gracias por querer contribuir. A continuación una guía rápida para colaborar de forma efectiva.

1. Forkea el repositorio y crea una rama descriptiva: `git checkout -b feat/mi-mejora`.
2. Mantén commits pequeños y con mensajes claros. Ejemplo: `feat: extraer CSS a styles.css`.
3. Abre un Pull Request contra la rama `main` del repositorio original.
4. Describe el cambio en el PR y, si aplica, añade capturas o pasos para reproducir.

Normas básicas

- Sigue una convención sencilla de commits: `feat:`, `fix:`, `chore:`, `docs:`.
- Evita cambios que rompan la visualización en mobile sin justificarlo.
- Si añades dependencias, explica por qué y añade instrucciones de instalación.

Reportar problemas

Usa Issues para bugs, solicitudes de características o preguntas. Provee:
- Pasos para reproducir
- Comportamiento esperado
- Comportamiento actual
- Capturas o logs si aplica

Código de conducta

Sé respetuoso y constructivo. Si algo te preocupa, abre un issue o contacta al mantenedor.
